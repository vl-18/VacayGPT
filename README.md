# VacayGPT 🌍

A simple **Streamlit-based travel recommendation app** that suggests destinations in India based on **travel dates and crowd preference**, with optional filters for **destination type and budget**. Destinations are ranked using a custom **Travel Score**.

## Features

* 📅 Travel date–based recommendations
* 👥 Crowd preference filtering
* 🏞 Optional destination type filter
* 💰 Optional budget filter
* 📊 Travel Score ranking (0–100)
* 🧭 Dataset of **popular and offbeat Indian destinations**
